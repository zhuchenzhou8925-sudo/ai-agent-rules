## 1. 一句话结论

晓羿协同平台可以被整理为“面向企业工程资料、图纸、模型和项目协作的一体化文件与图模协同平台”，重点价值是统一存储、权限管理、多端访问、二维/三维轻量化查看、任务协作、行业工具集成与本地化部署。

## 2. 平台定位

- 面向企业的文档、图纸、模型管理和应用工具。
- 支持组织搭建统一、高效、安全、智能的文件管理体系。
- 支持二维图纸、三维模型的网页端查看。
- 支持跨终端文件数据共享。
- 支持专业应用工具集成。
- 支持本地部署、开放接口和定制开发。

## 3. 服务对象与场景

| 对象 | 典型场景 |
|---|---|
| 设计方 | 设计协作、标注审查、出图查看、图模浏览 |
| 施工方 | 资料存储、图模浏览、问题反馈、项目协作 |
| 建设方 | 资料归档、项目协作、数字交付 |
| 运维方 | 资料存储、图模浏览、数字资产管理 |

## 4. 核心优势

| 优势 | 可复用表达 |
|---|---|
| 本地部署 | 数据完全私有，业务资料统一管理 |
| 开放接口 | 可与企业已有系统集成，支持定制化开发 |
| 多端支持 | Web、移动端、电脑客户端、专业软件插件端互联互通 |
| 性价比 | 以软件产品价格获得本地部署服务能力 |

## 5. 功能结构

### 5.1 文件管理

- 云端存储同步：支持大文件上传、下载和本地编辑后自动同步。
- 自动历史版本：自动保存近期修改版本，支持恢复或另存。
- 桌面日历：用于日程管理和复盘规划。
- 安全共享：生成安全分享链接，支持加密、访问人数控制和权限设置。
- 文件标签：支持自定义标签、批量标签和分类检索。
- 多端互联：支持电脑客户端、Web、专业软件插件端、移动小程序端。

### 5.2 项目协作

- 跨组织拉人建群：按部门、项目建立文件管理群，分配角色和权限。
- 文件保护：收藏重要文件，锁定正在编辑文件，降低版本冲突。
- 多维权限：项目级与文件级权限配置，覆盖查看、编辑、管理和分享。
- 任务管理：新建与发布任务，设置参与人和关注人，关联日历并跟踪进度。
- 项目动态：记录人员变化、文件增删改查、分享链接浏览下载转存情况。
- 远程收集：通过链接收集文件，并自动转存到网盘。

### 5.3 建筑设计行业应用

- 二维看图：CAD 图纸预览、圈注、标记、在线对比、在线校审。
- 三维看模：模型查看、视点、问题批注、漫游、剖切、测量。
- 合模：多专业、多格式模型云端整合、快速分享与查看。
- BIM 中心文件：基于云服务架构进行中心文件建模。
- 数据标准配置：面向报建报规、竣工交付、辅助审查等场景，支持模型数据标识配置与参数赋值。
- 问题报告：图纸/模型问题标记、问题传递、报告模板配置和一键生成。
- 智能构件库：支持本地 BIM 构件库和构件资产沉淀。

### 5.4 建筑施工行业应用

- AR 实模一致性对比：用于施工与验收环节的可视化参考。
- AR 实景动画展示：以移动设备播放工艺动画，辅助工法工艺和安全生产。
- 全流程统筹管理：Web 和移动端同步，支持多参与方共享、互通和透明。
- BIM + PM 集成：面向全过程资源管理和高效协同。
- 可视化进度与质量安全问题闭环管理。

### 5.5 工业行业应用

- 高保真工业模型查看：支持工业厂房、设施设备、材质纹理等模型展示。
- 工业设计模型轻量化查看：支持多类工业模型网页端快速查看。
- 模型解析与还原：支持不同颗粒度、格式和材质数据解析。
- 开放 API：支持图形引擎二次开发和接入企业自有平台。

### 5.6 数字孪生资产管理

- 各类文档资产数字化管理。
- 数字资产交付，辅助设施设备、文件资料全面管理。
- 三维可视化体验，集成多数据源。
- 数据、文档、图纸关联管理。
- 轻量化文档处理，一站式浏览模型和图纸。

### 5.7 政府招采服务

- 电子图纸检测压缩工具。
- 面向电子图纸规范检测、处理、导出流程。
- 支持本地部署，服务招投标主管部门电子化建设。

## 6. 可复用观点

1. “文件、图纸、模型、任务、权限”可以作为工程协同平台的五类基础对象。
2. 图模轻量化能力是工程资料管理从“存储型网盘”升级为“专业协同平台”的关键差异。
3. 本地部署与开放接口适合政企、设计院、施工企业、业主方等对数据私有化和系统集成要求较高的场景。
4. BIM 数据标准配置适合住建行业报建报规、辅助审查、竣工交付和企业数据标准落地。
5. 数字资产管理可以把文档、图纸、模型与设施设备管理关联起来，作为数字孪生资产底座。
