# Image / Diagram Placeholder Template

Use this template when a Word, PPT, Markdown, report, solution, or proposal output needs an image, diagram, architecture illustration, flowchart, cover visual, scene image, infographic, icon group, dashboard mockup, or prototype image.

Do not generate the image directly. Provide the placeholder and prompt only.

```markdown
【Image / Diagram Placeholder: {{name}}】

Usage position:
Describe the slide, page, chapter, section, or content module where this image should appear.

Expression purpose:
Explain what this image helps the reader understand or feel.

Core content:
List the main elements that should appear in the image or diagram.

Recommended layout:
Describe the preferred form, such as layered architecture diagram, horizontal process flow, vertical workflow, relationship map, scene visual, cover image, dashboard mockup, or infographic.

Style requirements:
Describe color, tone, aspect ratio, visual density, presentation style, and whether it should be formal, technical, executive, or lightweight.

AI image prompt:
Provide a complete prompt that can be copied into ChatGPT Image, image2, Midjourney, or another image-generation tool.
```
